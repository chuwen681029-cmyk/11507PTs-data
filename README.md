# 115-07 學生實習資料管理系統

純 HTML/CSS/JS 靜態網站，串接 Google Sheets + Google Drive API，
讓 17 位學生上傳期初／期末實習文件，並由管理者透過總表一覽上傳狀況。

## 檔案說明

| 檔案 | 說明 |
|---|---|
| `index.html` | 網站主頁面 |
| `style.css` | 視覺樣式（病歷夾風格設計） |
| `app.js` | 主程式邏輯（登入、讀寫 Sheet、上傳 Drive） |
| `students-data.js` | 17位學生名單、文件項目、登打欄位定義 |
| `config.js` | **需自行填寫**：Google Client ID / API Key / 試算表ID / 資料夾ID / 管理者Email |
| `setup-guide.md` | **完整設定步驟教學**（從建立 Google Cloud 專案到部署） |

## 快速開始

1. 詳閱 `setup-guide.md`，依步驟取得 Google API 金鑰並完成設定。
2. 編輯 `config.js`，填入您的金鑰、試算表ID、資料夾ID、管理者Email。
3. 將整個資料夾部署到 GitHub Pages 或任何靜態網站服務（或本機測試）。
4. 開啟網站，使用 Google 帳號登入即可使用。

### 期初分頁
- 上傳文件：體檢報告、保險證明、識別證、TMS教育訓練完成證明
- 登打資料：站別勾選（小兒／心肺）、Mac Address

### 期末分頁
- 上傳文件：學習護照、實習證明、實習成績、其它

### 角色
- **管理者**（`config.js` 中設定的 Google 帳號）：登入後看到全部17位學生的總表，
  可點擊各狀態圖示直接開啟 Drive 中的檔案，亦可代為上傳/編輯。
- **學生**：以本人 Gmail 登入，系統依顯示姓名自動對應名單，
  只會看到並能編輯自己的資料列與上傳區。

### 資料儲存
- 所有上傳的檔案會自動存入 Google Drive：
  `115-07 學生資料 / {學校}-{姓名} / 期初 或 期末 / 檔案`
- 上傳狀態與登打資料即時寫入「115-07 學生資料」試算表的「期初」「期末」分頁，
  作為總表資料來源。
